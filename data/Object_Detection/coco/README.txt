Solar Panel Damage Detection - Thermal Image Dataset: YOLO Version (compatible with v5 and v8)
==============================================================================================
This dataset contains over 1500 thermal labelled images of solar panels for object detection in 5 classes.
 
- Please split the dataset into train, test, valid folders as per your best knowledge.
 - Some Non annotated images are provided for testing, alternatively you can annotate and use it for training.
 - This dataset is compatible with YOLO X in case you are planning to use it.
